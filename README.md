## Getting Started

Install dependencies

```bash
pnpm install
```

### Development

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

Run the production build

```base
pnpm build
```

Run the production (local) server:

```base
pnpm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
